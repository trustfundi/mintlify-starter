---
title: "LOA management"
description: "Manage leave-of-absence requests for staff."
---

## LOA setup

Before staff can request LOA, configure:

- LOA manager role
- LOA active role
- Request channel or log channel
- Request cooldown

## Request flow

1. A user submits an LOA request.
2. Lumira sends the request to the LOA management area.
3. A user with the configured manager role approves or denies it.
4. Lumira gives the LOA role when the LOA starts.
5. Lumira DMs the user when the LOA begins and ends where DMs are available.

## End early

Managers can end an LOA early. Lumira logs the early end and removes the active
LOA role when configured.
