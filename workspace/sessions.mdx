## Sessions overview

Sessions help communities run trainings, patrols, meetings and hosted events.

## Dashboard start

When a session starts from the dashboard, Lumira posts the same Discord session
embed that the `/session` command uses. The embed includes the session name,
host, type, start time and join button when available.

## Attendance

Members can join the active session through the Discord button. Lumira stores
attendance and session logs so staff can review past activity.

## Session logs

The sessions page includes logs with:

- Session name
- Host
- Start and end time
- Duration
- Attendee count
