## Embed templates

The embed builder lets workspace staff create reusable Discord embeds for
announcements, session logs, workflows and other workspace messages.

## Supported fields

- Title
- Description
- Colour
- Thumbnail
- Image
- Author
- Footer
- Fields

## Reuse

Saved templates can be selected by supported features instead of writing raw JSON.
