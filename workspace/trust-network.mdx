## Public untrusted records

The Untrusted page shows approved global records reviewed by Lumira. Community
owners can use these records as risk signals before granting high-trust access.

## What may appear

- Roblox ID
- Roblox username or display name
- Discord ID when relevant
- Workspace context
- Severity
- Public evidence links when available

## Ban evasion

Pro workspaces can configure ban-evasion detection and review flagged
verifications from the dashboard.
