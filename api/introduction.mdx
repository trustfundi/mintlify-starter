## API version

Current public API version: `v1.1.0`

## Base URLs

```text
https://api.lumiraranking.co.uk
https://canary-api.lumiraranking.co.uk
```

Use production for stable systems and canary only when testing canary features.

## What the API is for

- Approved rank actions
- Workspace automation
- Roblox game integrations
- External logging or admin tools

<Warning>
  Do not use API access to bypass workspace permissions, Roblox group hierarchy,
  Discord permissions or Lumira plan limits.
</Warning>
