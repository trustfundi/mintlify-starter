---
title: "Promote command"
description: "Promote a Roblox group member from in-game."
---

## Command

```text
:promote username
```

The command sends a promote action to Lumira. Lumira validates the workspace,
API access, target user, ranking account and Roblox group hierarchy.

## Troubleshooting

- Confirm HTTP requests are enabled.
- Confirm the API key is valid.
- Confirm the target user is in the group.
- Confirm the ranking account can promote the target.
