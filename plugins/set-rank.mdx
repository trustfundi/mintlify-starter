---
title: "Set rank command"
description: "Set a Roblox group member to a specific rank from in-game."
---

## Command

```text
:setrank username rank
```

Use set rank when you need a specific target rank instead of the next promote or
demote step.

## Troubleshooting

- Use an actual Roblox role/rank that exists in the group.
- Check the role ID or rank number expected by your plugin version.
- Confirm the ranking account can assign the selected rank.
