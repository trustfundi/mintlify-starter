---
title: "Demote command"
description: "Demote a Roblox group member from in-game."
---

## Command

```text
:demote username
```

The command sends a demote action to Lumira. Lumira logs the action and uses the
connected ranking account to apply the Roblox rank change.

## Troubleshooting

- Confirm the target user is in the group.
- Confirm the ranking account can manage the target.
- Confirm the API key has not hit rate limits.
