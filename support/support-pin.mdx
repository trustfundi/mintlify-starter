## Why support asks for a PIN

The Support PIN lets Team Lumira verify a request without asking for private
credentials. It is safer than sending passwords, cookies, API keys or bot tokens.

## Find your PIN

1. Open the Lumira portal.
2. Go to **Account** or workspace account settings.
3. Copy **Support PIN**.
4. Send it only when Team Lumira asks for it.

## Include this context

- Support PIN
- Workspace ID
- Discord server ID
- Roblox group ID
- Feature affected
- Exact error text
- Screenshot or screen recording

<Danger>
  Never send Discord tokens, bot tokens, Roblox cookies, API keys, passwords or
  payment card details.
</Danger>
