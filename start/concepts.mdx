title: "Core concepts"
description: "Understand the objects Lumira uses across the workspace."
---

## Workspace

A workspace links one Discord server with Lumira configuration and Roblox group
operations. Workspace ownership stays attached to the Discord ID that created it.

## Ranking account

The ranking account is the Roblox account Lumira uses for rank actions. It is
connected in workspace settings through Roblox OAuth.

## Verified user

A verified user has linked a Roblox account to their Discord identity. Verification
data is stored so users do not need to reverify after the bot restarts.

## Role bind

A bind maps Roblox rank data to Discord roles. Binds keep role sync predictable
and reduce manual staff work.

## Support PIN

A Support PIN is a safe proof-of-account token found in account settings. Use it
when asking for help instead of sending private credentials.

## Plan

Core is the default plan. Pro unlocks advanced workspace operations such as LOA,
forms, workflows, embed builder, ban evasion detection, reputation, and custom bot
testing where enabled.
