## Core

Core is available by default and covers the essentials:

- Workspace setup
- Ranking
- Verification
- Role binds
- Sessions
- Moderation
- Member counter
- Presence feed
- API access with Core limits

## Pro

Pro can be assigned by Lumira staff or maintained through an active Stripe
subscription. Pro unlocks advanced operations:

- LOA management
- Forms builder
- Workflows and triggers
- Ban evasion detection
- Embed builder
- Reputation and leaderboards
- API logs
- Custom bot testing where enabled

## Billing behavior

If a subscription ends or payment fails, Pro-only features may be disabled after
the billing state updates.

<Tip>
  Check the billing page inside the portal to view subscription status,
  cancellation state, and remaining time.
</Tip>
