---
title: "Demote Plugin"
---

Promote users **down a rank** in-game using **Lumira’s API** alongside Basic Admin Essentials.

---

## **What this plugin does**

- **Demotes** a player to their previous Roblox group rank
- Sends **demotion** requests through the Lumira API
- Supports **demoting**:
  - A single user **(by username)**
  - Multiple users **_(e.g. trustfundi,abthecooldood,Twitch\_OliPlayz)_**

---

## **1) Open Roblox Studio**

- Launch **Roblox Studio**
- Open the place where **Basic Admin Essentials** is installed

---

## **2) Insert the Plugin ModuleScript**

1. In the **Explorer**, locate:\
   `BasicAdminEssentials → Plugins`
2. Create a **ModuleScript** inside the Plugins folder
3. Rename it to:
   ```text
   LumiraPromote
   ```
4. Delete all existing code inside the **ModuleScript**
5. Paste your Lumira **demote plugin code**

> Anything highlighted in green is where you must input the required information.

---

### **Plugin Code**

```text
COMING SOON
```

---

## **3) Configure the Plugin**

### **Required**

- `API_KEY` → your Lumira API key
- `GROUP_ID` → your Roblox group ID

---

## **4) Get your API Key**

- Join Lumira's server
- Generate your API key using the bot command _(e.g. `/apikey create`)_
- Copy your **API** key
- Paste it into:

```text
local API_KEY = 'API_KEY'
```

---

## **5) Ensure Roblox Settings Are Correct**

- Go to **Game Settings → Security**\\
- ✅ Enable **Allow HTTP Requests**\\

Also make sure:

- Your Roblox Group ID is correct
- Your **API Key** is valid
- The group has a proper rank structure
- Your system has permission to demote users

---

## **6) How to Use the Command (In-Game)**

### **Demote One Player**

Example:

```text
:demote username
```

> The command name depends on the **ModuleScript** name.

---

## **Troubleshooting**

### **“API error” / Demotions failing**

- Ensure **HTTP Requests** are enabled
- Verify your **API key** is correct
- Make sure **Lumira services** are online
- Confirm your **group ID** is valid
- Ensure your system has permission to change ranks
