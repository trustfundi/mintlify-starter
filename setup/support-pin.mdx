## What the Support PIN is

The Support PIN is a temporary proof-of-account value from your Lumira account
settings. It helps Team Lumira verify requests without asking for passwords,
tokens, cookies, or private credentials.

## Where to find it

1. Open the Lumira portal.
2. Go to **Account** or workspace account settings.
3. Find **Support PIN**.
4. Copy the PIN when support asks for it.

## What to include with support requests

- Support PIN
- Workspace ID or Discord server ID
- Roblox group ID
- Feature or page affected
- Exact error message
- Screenshot or short recording if available
